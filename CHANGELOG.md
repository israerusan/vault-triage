# Changelog

All notable changes to Note Doctor are documented here. The format follows
[Keep a Changelog](https://keepachangelog.com/), and the project adheres to
[Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-07-05

### Added

- Initial release.
- Vault scan with five built-in issue detectors: stale notes, thin notes, orphan
  notes, missing required properties, and draft markers.
- Dashboard with issue counts by category, last-scan summary, and quick filters.
- Results list with filtering, sorting, per-row actions (open, reveal, ignore,
  exclude, mark reviewed).
- Review queue for working through flagged notes one at a time.
- Configurable thresholds, required properties, draft markers, and exclusions.
- Pro (one-time unlock): saved scan profiles, custom rules, safe bulk actions,
  advanced review workflows, severity tuning, and Markdown report export.
- Offline Ed25519 license-key verification. No account, no server.
